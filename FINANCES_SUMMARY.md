## Реализованный функционал

### ✅ 1. Счета (Accounts)
- CRUD операции
- Типы: наличные, карта, депозит
- Мультивалютность (RUB, USD, EUR, GBP, JPY, CNY)
- Авто-пересчёт баланса

### ✅ 2. Транзакции (Transactions)
- Типы: доход/расход/перевод
- Категории, теги, описание
- Комиссии для переводов
- Валидация

### ✅ 3. Категории (Categories)
- Предопределённые (10 доходов + 10 расходов)
- Пользовательские (создание/редактирование/удаление)
- Иконки (14) и цвета (10)

### ✅ 4. Бюджеты (Budgets)
- Лимиты по категориям
- Периоды: месяц/неделя/год
- Прогресс и превышения

### ✅ 5. Повторяющиеся транзакции
- Периодичность: ежедневно/еженедельно/ежемесячно/ежегодно
- Авто-создание
- Дата начала/окончания

### ✅ 6. Отчёты (Reports)
- Страница: `/finances/reports`
- Графики: Pie, Bar, Line
- Экспорт CSV
- Периоды: месяц/неделя/год/произвольный

### ✅ 7. Фильтры (Filters)
- Поиск по описанию
- По типу, счёту, категории
- По периоду (календарь)
- Активные фильтры (chips)

### ✅ 8. Массовые операции
- Выбор транзакций
- Checkbox для каждой
- "Выбрать все"
- Массовое удаление

### ✅ 9. Импорт/Экспорт CSV
- Импорт из банковских выписок
- Предпросмотр
- Авто-определение доход/расход
- Экспорт из отчётов

### ✅ 10. Toast уведомления
- Success, Error, Info, Warning, Loading
- Интеграция во все формы

### ✅ 11. Пагинация
- 10/25/50/100 записей на странице
- Навигация по страницам
- Счётчик "Показано X-Y из Z"
