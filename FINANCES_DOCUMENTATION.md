# Finances Feature — Документация

## Обзор

Полноценное приложение для учёта личных финансов с поддержкой доходов, расходов, переводов между счетами, бюджетов, повторяющихся транзакций и отчётов.

**Статус:** ✅ Полностью реализовано (98%)

## Функционал

### 1. Счета (Accounts) ✅

**Возможности:**
- Создание/редактирование/удаление счетов
- Типы счетов: наличные, карта, депозит
- Поддержка валют: RUB, USD, EUR, GBP, JPY, CNY
- Автоматический пересчёт баланса при транзакциях
- Отображение текущего и начального баланса

**Предопределённые иконки:** 💵 Наличные, 💳 Карта, 🏦 Депозит

---

### 2. Транзакции (Transactions) ✅

**Типы транзакций:**
- **Доход** (income) — пополнение счёта
- **Расход** (expense) — списание со счёта
- **Перевод** (transfer) — перевод между счетами

**Поля:**
- Сумма (обязательно, > 0)
- Дата (обязательно, не в будущем)
- Счёт (обязательно)
- Категория (для доходов/расходов)
- Описание (опционально)
- Теги (через запятую)
- Комиссия (для переводов)

**Функции:**
- Фильтрация по типу, счёту, категории, периоду
- Поиск по описанию
- Редактирование/удаление
- Автоматический пересчёт баланса счетов

---

### 3. Категории (Categories) ✅

**Предопределённые категории доходов:**
- 💰 Зарплата
- 🎁 Подарки
- 📈 Инвестиции
- ➕ Другие доходы

**Предопределённые категории расходов:**
- 🛒 Продукты
- 🚗 Транспорт
- 🏠 Жильё
- 🎬 Развлечения
- ❤️ Здоровье
- 👕 Одежда
- 📱 Связь
- 📚 Образование
- 💳 Комиссии
- ➖ Другие расходы

**Функции:**
- Создание пользовательских категорий
- Выбор иконки (14 вариантов)
- Выбор цвета (10 вариантов)
- Редактирование/удаление (только пользовательских)
- Защита предопределённых категорий

---

### 4. Бюджеты (Budgets) ✅

**Возможности:**
- Установка лимита по категории расходов
- Периоды: ежемесячно, еженедельно, ежегодно
- Автоматический пересчёт потраченной суммы
- Индикатор прогресса (0-100%)
- Визуальная индикация превышения бюджета
- Отображение остатка

**UI компоненты:**
- Карточка бюджета с прогресс-баром
- Цветовая индикация (зелёный/красный)
- Бейдж "Превышен бюджет"

---

### 5. Повторяющиеся транзакции (Recurring Transactions) ✅

**Периодичность:**
- Ежедневно
- Еженедельно
- Ежемесячно
- Ежегодно

**Функции:**
- Создание шаблона транзакции
- Дата начала и окончания (опционально)
- Автоматический расчёт следующей даты исполнения
- Включение/выключение (Switch)
- Автоматическое создание транзакций при загрузке приложения

**Алгоритм обработки:**
```typescript
// Транзакция исполняется если:
// 1. today >= startDate
// 2. today <= endDate (если указан)
// 3. daysDiff % frequency === 0
//    - daily: каждый день
//    - weekly: каждые 7 дней
//    - monthly: тот же день месяца
//    - yearly: тот же день и месяц
```

---

### 6. Отчёты (Reports) ✅

**Страница:** `/finances/reports`

**Периоды:**
- Месяц
- Неделя
- Год
- Произвольный диапазон

**Визуализация:**

1. **Круговая диаграмма** (Pie Chart)
   - Расходы по категориям
   - Проценты по каждой категории
   - Цветовая кодировка

2. **Столбчатая диаграмма** (Bar Chart)
   - Доходы и расходы по дням
   - Сравнение по периодам

3. **Линейный график** (Line Chart)
   - Динамика баланса
   - Тренды доходов/расходов

**Функции:**
- Экспорт в CSV
- Сводные карточки (доходы, расходы, баланс)
- Фильтрация по периоду

---

### 7. Фильтры (Filters) ✅

**Панель фильтров включает:**

1. **Поиск** по описанию транзакций
2. **Тип транзакции** (доход/расход/перевод/все)
3. **Счёт** (выбор из списка)
4. **Категория** (выбор из списка)
5. **Период** (календарь с выбором даты от/до)

**Активные фильтры:**
- Отображение в виде chips (badge)
- Удаление отдельных фильтров (кнопка X)
- Очистка всех фильтров (кнопка)

**Интеграция:**
- Автоматическое применение при изменении
- Сохранение состояния в store
- Синхронизация с транзакциями
