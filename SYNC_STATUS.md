## Проверка работы

### 1. Тест API
```bash
curl http://localhost:3000/api/sync
```

**Ожидаемый ответ:**
```json
{
  "status": "ok",
  "timestamp": "2026-03-09T...",
  "version": "1.0.0",
  "configured": true
}
```

### 2. Тест синхронизации
```bash
curl -X POST http://localhost:3000/api/sync \
  -H "Content-Type: application/json" \
  -d "{\"changes\":[],\"lastSync\":\"2026-01-01T00:00:00Z\"}"
```

**Ожидаемый ответ:**
```json
{
  "success": true,
  "changes": [],
  "processed": [],
  "lastSync": "2026-03-09T..."
}
```

## Следующие шаги

Синхронизация полностью готова! При запуске приложения:

1. **Локальные изменения** автоматически отправляются в Supabase
2. **Удалённые изменения** загружаются каждые 30 секунд
3. **Offline режим** работает - изменения кэшируются в IndexedDB
4. **При восстановлении соединения** происходит автосинхронизация
